# 🌐 IOTHON - Metaverse IoT Hackathon

IOTHON is a metaverse-based hackathon organized by Enthiran Robotics Club,  
focused on building innovative IoT solutions in a virtual collaborative environment.

## 🚀 About the Event
IOTHON brings together developers, designers, and innovators to create smart solutions  
that connect the physical and digital worlds using IoT and emerging technologies.

## 🎯 Themes
- Smart Cities  
- Healthcare IoT  
- Industrial Automation  
- Metaverse + IoT Integration  
- Open Innovation
